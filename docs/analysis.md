# Анализ текущих сайтов Сергея Дудченко

## Сайт 1: tt.dudc-sergej.ru (основной турнирный)

### Технологии
- **CMS**: WordPress 6.9.1
- **Тема**: boxing-martial-arts (Themespride)
- **Аналитика**: WP Statistics + Яндекс.Метрика
- **Реклама**: Яндекс RTB (Adsense)

### Структура страниц

#### 1. Главная страница
- Заголовок: "ТУРНИР 'МИРОК НАСТОЛЬНОГО ТЕННИСА' И НЕ ТОЛЬКО"
- Подзаголовок: "Любительский рейтинговый турнир по настольному теннису"
- Лента последних турниров (карточки с фото, датой, автором)
- Боковая панель: календарь, архивы, последние записи, комментарии
- Пагинация: 5 страниц турниров
- Фоновое изображение: cropped-glavnaya1.jpg

#### 2. Регистрационная форма (/регистрационная-форма/)
- Поля: Фамилия И.О., TTW-Рейтинг (число), Взнос оплачен (чекбокс)
- Простая HTML-форма без плагина (самописная)
- Ширина 350px, зелёная кнопка

#### 3. Участники турнира (/участники-турнира-2/)
- Список участников на ближайший турнир (25.02.2026)
- Мин: 5 чел., макс: 16 чел.
- Время начала: 10:30
- Данные, вероятно, загружаются динамически

#### 4. О турнире (/о-турнире/)
- Описание: любительский рейтинговый, личная инициатива
- Формат: утренние соревнования в будни
- Условия: открыт для всех уровней, минимальный взнос (аренда столов)
- Рейтинги: TTW и RTTF
- Награды: медали, электронные грамоты, фото на сайте
- Информация: сайт + Телеграм-канал + трансляции

#### 5. Регламент (/sample-page/)
- Официальный регламент турнира
- Место: ТРЦ "Палас", Симферополь
- График: по средам 10:30-14:00
- Макс. 16 участников
- Взнос ~500 руб. (только аренда)
- Некоммерческий, без призового фонда
- Правила по нормативам Минспорта РФ

#### 6. Правила н/тенниса (/правила-настольного-тенниса/)
- Полный текст официальных правил (Приказ Минспорта от 19.12.2017 №1083)
- Разделы: виды соревнований, правила игры, оборудование, судейство
- Очень объёмный документ

#### 7. Тесты/Викторины (/тесты-викторины-опросы-голосования/)
- Мифы о настольном теннисе
- Вредные советы (юмор)
- Интересные факты
- Тест на знание правил (20 вопросов, порог 18/20)
- Тест для игроков и тренеров
- Первое прохождение бесплатно, далее 100 руб.

#### 8. Тренер (/тренер-по-настольному-теннису/)
- Образование: "Теория и методика учебно-тренировочного процесса по НТ"
- Для начального и среднего уровня
- Обучение: базовые и продвинутые техники, разные хваты, игра двумя руками
- Оборудование: теннисная пушка
- Место: ТРЦ "ПАЛАС", 3 этаж, "Ping Pong Club"
- График: ежедневно 10:00-22:00 по записи
- Первое занятие бесплатно

## Проблемы текущих сайтов

### UX/UI
1. **Два отдельных сайта** — путаница для пользователей
2. **Устаревший дизайн** — тема boxing-martial-arts не соответствует теннисной тематике
3. **Перегруженная навигация** — слишком много пунктов, неочевидные названия
4. **Мелкий логотип** (25px) — плохо виден
5. **Реклама Яндекс RTB** — отвлекает от контента
6. **Нет единого стиля** между двумя сайтами

### Контент
1. **URL на кириллице** — проблемы с совместимостью и SEO
2. **Sample Page** используется как Регламент — непрофессионально
3. **Огромная страница правил** — не адаптирована для чтения
4. **Тесты за деньги** — может отпугивать пользователей

### Техника
1. **Изображения 3000x1500px** — не оптимизированы, медленная загрузка
2. **Нет HTTPS** на втором сайте (http://tt.sergey-dudchenko.ru)
3. **Нет адаптивных изображений** (srcset)
4. **WordPress с тяжёлыми темами** — избыточно для простого сайта
