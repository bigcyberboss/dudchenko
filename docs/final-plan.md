# ФИНАЛЬНЫЙ ПЛАН: Объединение 3-х сайтов в 1 современный

## Проблема сейчас: 3 разрозненных сайта

| Сайт | CMS | Назначение | Проблемы |
|------|-----|-----------|----------|
| tt.dudc-sergej.ru | WordPress (тема boxing) | Турнир "Мирок" | Устаревший дизайн, тема для бокса, реклама Яндекса |
| tt.sergey-dudchenko.ru | WordPress (тема Globetrotter) | Личный блог о НТ + субботние турниры | Дублирует контент, hero видео не грузится |
| sergey-dudchenko.ru | Typesetter CMS | Портфолио (библиотека, легенды, курсы) | Не связан с теннисом, старый дизайн |

**Люди путаются** между тремя сайтами. Навигация ведёт с одного на другой и обратно.

---

## Решение: 1 сайт — всё в одном месте

### Концепция дизайна

**Стиль**: Современный спортивный, чистый, энергичный
**Вдохновение**: SPIN Ping Pong (hero-видео + booking), Nike (энергия + минимализм)
**Целевая аудитория**: Люди 40-70 лет — крупные шрифты, простая навигация, высокий контраст

### Принципы UI/UX для возрастной аудитории
- Базовый шрифт: **18px** (не меньше 16px нигде)
- Кнопки: минимум **48x48px** (удобно для тапа)
- Контрастность: **4.5:1 минимум** (WCAG AA)
- Навигация: **максимум 6 пунктов** в главном меню
- Иконки: **всегда с текстовой подписью**
- Язык: простой, понятный, без жаргона
- Grid-раскладка для лёгкого сканирования

### Цветовая палитра

```
Основной:    #1B5E20 (тёмно-зелёный — стол)
Акцент:      #FF6D00 (оранжевый — мяч)
Фон светлый: #FAFAFA
Фон тёмный:  #0A1628 (для hero и footer)
Текст:       #212121
Текст light:  #E0E0E0 (на тёмном фоне)
Успех:       #4CAF50
Ссылки:      #1565C0
```

### Типографика
- **Заголовки**: Montserrat Bold (энергичный, спортивный)
- **Текст**: Open Sans Regular (легко читается)
- **Размеры**: h1=48px, h2=36px, h3=28px, body=18px, small=16px

## Структура сайта (8 страниц)

### 1. ГЛАВНАЯ (index.html)
```
┌─────────────────────────────────────────────┐
│ HEADER: Логотип "МИРОК" + Меню + Кнопка    │
│         "Записаться на турнир"               │
├─────────────────────────────────────────────┤
│ HERO: Фото/видео Сергея за столом           │
│   «МИРОК НАСТОЛЬНОГО ТЕННИСА»               │
│   Любительский рейтинговый турнир            │
│   [Ближайший турнир] [О тренере]            │
├─────────────────────────────────────────────┤
│ БЛИЖАЙШИЙ ТУРНИР: Дата + Обратный отсчёт    │
│   Место: ТРЦ "Палас" | Время: 10:30        │
│   Участников: 8/16 | [Зарегистрироваться]   │
├─────────────────────────────────────────────┤
│ О ТУРНИРЕ: 3 карточки с иконками            │
│   [Открыт для всех] [Рейтинговый] [Недорого]│
├─────────────────────────────────────────────┤
│ ПОСЛЕДНИЕ ТУРНИРЫ: 3 карточки с баннерами   │
│   Результаты + победители + [Подробнее]     │
├─────────────────────────────────────────────┤
│ ТРЕНЕР: Мини-профиль + фото + [Подробнее]   │
│   "Первое занятие бесплатно!"               │
├─────────────────────────────────────────────┤
│ FOOTER: Контакты, соцсети, партнёры         │
│   YouTube | VK | Telegram | Email           │
│   [TTW] [RTTF] логотипы                     │
│   © Дудченко С.В.                           │
└─────────────────────────────────────────────┘
```

### 2. О ТУРНИРЕ (about.html)
- История: как всё начиналось (личная история Сергея)
- Формат турнира (еженедельно, среда, 10:30-14:00)
- Условия участия (открыт для всех, взнос ~500 руб.)
- Рейтинговые системы (TTW + RTTF с виджетами/ссылками)
- Награды: медали, грамоты, фото на сайте
- Фотогалерея с лайтбоксом
- Встроенные видео с YouTube канала

### 3. ТУРНИРЫ И РЕЗУЛЬТАТЫ (tournaments.html)
- Таблица участников БЛИЖАЙШЕГО турнира (с TTW-рейтингом)
- Статус: "Осталось N мест из 16"
- Архив результатов по датам (аккордеон/табы)
- Каждый турнир: дата, участники, результаты 1-2-3 место, баннер
- Фильтр по месяцам

### 4. РЕГИСТРАЦИЯ (register.html)
- Красивая форма на фоне с размытием (glassmorphism)
- Поля: ФИО, TTW-рейтинг, Telegram/телефон, чекбокс оплаты
- Валидация в реальном времени
- Правила участия (кратко, ссылка на полные)
- Контакт для вопросов

### 5. ПРАВИЛА (rules.html)
- **Регламент турнира** — компактные аккордеоны по разделам
- **Правила НТ** — ключевые моменты с иллюстрациями
- Ссылка на полный PDF (Минспорт РФ)
- FAQ — частые вопросы

### 6. ТРЕНЕР (coach.html)
- **Hero**: фото Сергея за столом + краткое описание
- **История**: начал после 50, похудел со 127 кг, играет двумя руками
- **Квалификация**: галерея дипломов и сертификатов (лайтбокс)
  - Диплом тренера НТ
  - Приложение к диплому
  - Сертификат профстандарта
  - Спортивная награда профсоюза
- **Уникальность**: 2 рейтинга TTW (правая + левая рука)
- **Услуги**: техника, хваты, теннисная пушка, подбор инвентаря
- **Расписание**: ежедневно 10:00-22:00, первое занятие бесплатно
- **Видео**: встроенные с YouTube
- **Форма записи** на занятие
- **Место**: карта / адрес ТРЦ "Палас"

### 7. ИНТЕРЕСНОЕ (fun.html)
- Тесты и викторины (интерактивные, встроенные)
- Мифы о настольном теннисе
- Вредные советы (юмор)
- Интересные факты
- Полезные ссылки

### 8. ОБ АВТОРЕ (author.html) — опционально
- Портфолио (если Сергей хочет сохранить)
- Ссылки на другие проекты (библиотека, легенды Крыма)
- Можно сделать лёгкой страницей-визиткой

## Навигация (6 пунктов + кнопка)

```
[Главная] [О турнире] [Турниры] [Правила] [Тренер] [Интересное]  [ЗАПИСАТЬСЯ]
```

**Мобильное меню**: гамбургер → полноэкранное меню с крупными пунктами

## Технический стек

- **HTML5** (семантическая разметка, ARIA)
- **CSS3** (переменные, Grid, Flexbox, анимации, @media)
- **Vanilla JavaScript** (без фреймворков — для простоты и скорости)
- **Google Fonts** (Montserrat + Open Sans)
- **GitHub Pages** для хостинга демо

### Оптимизация
- Изображения: WebP, lazy loading, srcset
- CSS: переменные для темизации, минимум повторов
- JS: defer, модульный, < 50KB
- Цель: **Lighthouse 95+**
