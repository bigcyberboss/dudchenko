# План разработки нового сайта "Мирок настольного тенниса"

## Концепция
Один современный, красивый, быстрый сайт вместо двух устаревших WordPress-сайтов.
Статический сайт (HTML/CSS/JS) — быстрый, безопасный, бесплатный хостинг на GitHub Pages.

## Структура сайта (7 страниц)

### 1. Главная (index.html)
- Hero-секция с эффектным фото/видео и заголовком турнира
- Ближайший турнир (дата, время, место) с обратным отсчётом
- Быстрая навигация по разделам (карточки)
- Последние результаты турниров (3 шт.)
- Контактная информация + ссылки на соцсети

### 2. О турнире (about.html)
- История турнира и автора
- Формат и условия участия
- Фотогалерея
- Рейтинговые системы (TTW, RTTF)
- Награды и достижения

### 3. Участники и результаты (participants.html)
- Таблица участников ближайшего турнира
- Архив результатов прошлых турниров
- Мини-профили участников (опционально)
- Фильтры по дате

### 4. Регистрация (registration.html)
- Красивая форма регистрации
- Поля: ФИО, TTW-рейтинг, контакт (Telegram/телефон)
- Статус оплаты
- Правила участия (кратко)

### 5. Правила и регламент (rules.html)
- Регламент турнира (компактно, с аккордеонами)
- Основные правила настольного тенниса (ключевые моменты)
- Ссылка на полный текст правил (PDF)
- FAQ по частым вопросам

### 6. Тренер (coach.html)
- Профиль тренера с фото
- Квалификация и опыт
- Услуги и программа обучения
- Стоимость и расписание
- Форма записи на занятие
- Отзывы учеников

### 7. Блог/Новости (blog.html)
- Тесты и викторины
- Интересные факты
- Мифы о настольном теннисе
- Новости турнира

## Технический стек

### Фронтенд
- **HTML5** — семантическая разметка
- **CSS3** — кастомные свойства (переменные), Grid, Flexbox, анимации
- **JavaScript** — ванильный, без фреймворков
- **Шрифты**: Google Fonts (Montserrat + Open Sans)

### Оптимизация
- Изображения: WebP формат, lazy loading, srcset для адаптивности
- CSS: минификация, критический CSS инлайн
- JS: минификация, defer/async загрузка
- Цель: Lighthouse score > 90

### Адаптивность
- Mobile-first подход
- Брейкпоинты: 320px, 768px, 1024px, 1440px
- Удобные кнопки для тач-устройств (мин. 44x44px)

### Доступность
- WCAG 2.1 AA
- Базовый шрифт: 18px (для возрастной аудитории)
- Контрастность: минимум 4.5:1
- Все изображения с alt-текстами
- Навигация с клавиатуры
