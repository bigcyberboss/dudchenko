# Тренды веб-дизайна 2025-2026 для спортивного сайта

## Применимые тренды

### 1. Органический дизайн
- Плавные линии и мягкие градиенты вместо жёстких сеток
- Скруглённые углы, естественные формы
- Тёплые, живые цвета

### 2. Микроанимации
- Ненавязчивые hover-эффекты на кнопках и карточках
- Плавные переходы между секциями
- Анимация при прокрутке (scroll-triggered)
- Функциональные, а не декоративные

### 3. Типографика как герой
- Крупные заголовки с характером
- Контрастные шрифтовые пары (заголовок + текст)
- Читаемость на первом месте

### 4. Mobile-First
- Дизайн сначала для телефона, потом для десктопа
- Большие кнопки, удобные для тапа
- Быстрая загрузка на мобильном интернете

### 5. Производительность
- Лёгкие страницы (< 1 МБ)
- Оптимизированные изображения (WebP, lazy loading)
- Минимум внешних зависимостей

### 6. Доступность (Accessibility)
- Контрастные цвета (WCAG AA минимум)
- Правильная иерархия заголовков
- ARIA-метки
- Крупный шрифт (базовый 16-18px)
- Особенно важно для возрастной аудитории!

### 7. Glassmorphism (стеклянный эффект)
- Полупрозрачные карточки с размытием фона
- Создаёт глубину и современный вид
- Использовать дозировано

### 8. Яркие насыщенные цвета
- "Допаминовый дизайн" — яркие, энергичные палитры
- Неоновые акценты на тёмном или светлом фоне
- Градиенты вместо плоских заливок

## Цветовая палитра для настольного тенниса

### Вариант 1: Энергичный
- **Основной**: #1B5E20 (тёмно-зелёный, цвет стола)
- **Акцент**: #FF6D00 (оранжевый, цвет мяча)
- **Фон**: #FAFAFA (почти белый)
- **Текст**: #212121 (почти чёрный)
- **Вторичный**: #0D47A1 (синий, сетка)

### Вариант 2: Современный тёмный
- **Основной фон**: #0A1628 (тёмно-синий)
- **Акцент**: #00E676 (ярко-зелёный)
- **Вторичный акцент**: #FF9100 (оранжевый)
- **Текст**: #E0E0E0 (светло-серый)
- **Карточки**: rgba(255,255,255,0.05) (стеклянный эффект)

### Вариант 3: Чистый и светлый
- **Фон**: #FFFFFF
- **Основной**: #2E7D32 (зелёный)
- **Акцент**: #F57C00 (оранжевый)
- **Серый**: #F5F5F5 (секции)
- **Текст**: #333333
